Using Advanced Filters

Filters allow you tune your search to return exactly what interests you.

Depending on the width of your internet browser, none or some of the filters will be displayed in the search menu at the top. You can access all of the hidden filters by click on the 'Filters ...' button in the search menu. The filters you can specify include:

  1. Start Date and End Date fields. The date picker will automatically constrain your selection to a valid range for the selected dataset. Seasonal Search allows constraining the search to certain annual periods within an overall range of dates. Click the Seasonal Search toggle and additional options will appear, allowing you to enter an overall date range (Start Date/End Date) and the seasonal range (Season Start Day/Season End Day).
  2. {{ ' ADDITIONAL_FILTERS ' | translate }} allows for additional parameters to be applied to narrow your search and reduce the number of results. Not all filters will be available for all datasets, and this list can change.
    1. File Type
    2. Subtype
    3. Beam Mode
    4. Polarization
    5. Direction
    6. {{ 'CAMPAIGN_SELECTOR' | translate }}
