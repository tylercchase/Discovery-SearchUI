Get an Earthdata Login

If you do not already have one, register for a free Earthdata Login account.

  1. An easy way to register for an Earthdata account is to click the "Sign in" icon in the upper-right corner of Vertex, then select the "REGISTER" button and fill-in the necessary information on the subsequent form. Note that you can resize the form if desired.
  2. Or, you can simply go to https://urs.earthdata.nasa.gov/users/new and fill-in the form to register.
