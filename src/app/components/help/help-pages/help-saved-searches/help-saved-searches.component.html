Save a Search

Make sure you are logged into Vertex. Searches cannot be saved and search history cannot be accessed unless you are logged in.

To save a search follow these steps:

  1. Define a search you would like to save. Enter the search type, dataset, area of interest and all of the other search filters of interest to you to create the search exactly as you like it. You can execute the search and verify it performs as you expect.
  2. Click the arrow down icon on the 'Search' button to open the search menu.
  3. Select 'Save Search' from the search menu.
  4. Enter a name for your search in the Save Search dialog.

You have now saved your search and can execute it at anytime.

To view your saved searches follow these steps:

  1. Make sure you are logged into Vertex.
  2. Click the arrow down icon on the 'Search' button to open the search menu.
  3. Select 'Saved Searches...' from the search menu.

You can view, execute and delete your saved searches from the dialog that appears.

To view your search history follow these steps:

  1. Make sure you are logged into Vertex.
  2. Click the arrow down icon on the 'Search' button to open the search menu.
  3. Select 'Search History...' from the search menu.

You can view and execute searches you have done from the dialog that appears.
